После перезапуска Claude Code спросите:

   "Какие у тебя есть инструменты для работы с земельными участками?"

Claude должен показать 16 инструментов:
   • cadastre_get_boundaries, cadastre_get_info, cadastre_search_by_address
   • geocode_address_to_coords, geocode_coords_to_address, geocode_validate_address
   • infrastructure_find_nearby, infrastructure_calculate_distances, infrastructure_get_satellite_image
   • geometry_calculate_area, geometry_check_intersection, geometry_measure_distance, geometry_buffer
   • visualization_generate_static_map, visualization_export_geojson, visualization_export_to_json

Попробуйте:

1. "Найди координаты адреса Москва, Красная площадь, 1"
   → Использует geocode_address_to_coords

2. "Покажи информацию о участке 77:01:0005001:1234"
   → Использует cadastre_get_info

3. "Найди все школы в радиусе 2км от координат 55.7558, 37.6173"
   → Использует infrastructure_find_nearby

4. "Рассчитай расстояние между точками (55.75, 37.61) и (55.76, 37.62)"
   → Использует geometry_measure_distance

5. "Создай спутниковый снимок для координат 55.7558, 37.6173"
   → Использует infrastructure_get_satellite_image

❓ Сервер не появляется в Claude?
   → Проверьте ~/.claude/mcp-servers.json
   → Полностью перезапустите Claude Code

❓ Ошибка "Module not found"?
   → cd construction-maps-mcp
   → /usr/local/bin/python3.14 -m pip install --user -e .

❓ Медленная работа?
   → Первый запрос к API медленный
   → Повторные запросы из кеша быстрые

❓ Нужны детальные логи?
   → Откройте .env
   → Установите LOG_LEVEL=DEBUG
